This text is part of an ongoing research and writing project.

This repository collects notes, articles, and early explorations related to
software engineering and data analysis applied to motorsport and racing simulation.

The goal is not to build a finished product immediately, but to:
- understand how data is used in motorsport contexts
- explore software design choices for analysis tools
- connect theoretical computer science with applied motorsport problems

  Drafts and published versions of technical articles.

- [notes](\notes\README.md)
