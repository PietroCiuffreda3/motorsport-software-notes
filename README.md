This repository is a Jekyll site that collects notes, articles, and early explorations
related to software engineering and data analysis applied to motorsport and racing simulation.

The goal is not to build a finished product immediately, but to:
- understand how data is used in motorsport contexts
- explore software design choices for analysis tools
- connect theoretical computer science with applied motorsport problems

## Contents

- `_posts/`
  Long-form articles (published).

- `_projects/`
  Project pages / ongoing work.

- `api/`
  JSON endpoints generated by Jekyll.

- `idea.md`
  Open questions and topics to explore.

## Local development

```bash
bundle install
bundle exec jekyll serve
```

Then open the printed local URL.
